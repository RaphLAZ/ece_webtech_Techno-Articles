## Getting Started

# Open a terminal 

* Clone our repository :

```
git clone https://github.com/RaphLAZ/ece-webtech-gr02-04/tree/main
```

* Go in the folder :
```
cd ./ece-webtech-gr02-04
```

* Download all related packages :
```
npm install
```
* Run the app :
```
npm run dev
```
